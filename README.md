# College Placement Portal

## Overview
College Placement Portal is a web-based application that helps students apply for placement opportunities online.

## Features
- Student Login
- Company Job Listings
- Resume Upload
- Department Selection
- Application Dashboard
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js

## How to Run
1. Download the project
2. Open index.html in browser
